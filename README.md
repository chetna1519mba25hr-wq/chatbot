Chetna HR Assistant Chatbot

## Overview

This project is a simple **HR Assistant Chatbot UI** built using HTML, CSS, and JavaScript. It simulates a conversational interface where users can interact with a chatbot designed to answer HR-related queries.

The chatbot provides a user-friendly interface with real-time messaging, making it suitable for basic automation and learning chatbot development.

## Features

*  Interactive chat interface
*  User input with Enter key support
*  Automated bot responses
*  Typing delay effect for realism
*  Auto welcome message
*  Clean and responsive UI design

---

##  Project Structure

### 1. Frontend UI

* Built using **HTML & CSS**
* Simple and clean chatbot layout

### 2. JavaScript Functionality

* Handles user input
* Sends messages dynamically
* Generates bot responses
* Adds typing delay effect

---

## How It Works

1. User enters a message in the input box
2. Message is displayed in the chat
3. Bot processes input using predefined logic
4. Bot replies with a response after a short delay
5. Chat continues interactively

---

##  Key Functions

* `sendMessage()` → Handles user input
* `addMessage()` → Displays messages in chat
* `getBotResponse()` → Generates bot replies
* Event listener → Sends message on Enter key
